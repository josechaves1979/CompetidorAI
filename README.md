# CompetitorAI

## Sobre o projeto

O CompetitorAI é uma plataforma baseada em Inteligência Artificial desenvolvida para auxiliar profissionais de Inteligência de Mercado na análise competitiva e monitoramento da concorrência.

O sistema foi concebido para:

- Monitorar preços dos concorrentes;
- Identificar promoções e campanhas comerciais;
- Comparar posicionamento de produtos;
- Gerar insights estratégicos automaticamente;
- Auxiliar na tomada de decisão baseada em dados.

## Tecnologias utilizadas

- Python
- Inteligência Artificial
- Streamlit
- Pandas
- GitHub
- Ciência de Dados

## Funcionalidades

- Cadastro dos concorrentes;
- Comparação de preços;
- Monitoramento do mercado;
- Dashboard gerencial;
- Geração automática de insights;
- Relatórios executivos.

- Análise preditiva de preços.
